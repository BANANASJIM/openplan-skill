# OpenPlan Align

Clarify goal/intent before action using OpenPlan philosophy.

Workflow:

1. Restate the user's request in one paragraph.
2. Separate known facts from assumptions.
3. Use the smallest useful depth of the 7-layer funnel:
   - WHY
   - WHO/WHERE
   - WHAT
   - HOW macro
   - RISK
   - VERIFY
   - DETAIL
4. Identify blind spots and human-owned decisions.
5. Recommend one next route: ask via the human-facing coordinator, research, design, implement, review, or stop.

Do not implement or review changes in this command. Ask at most 1-3 questions only if you are the human-facing coordinator; delegated agents should report `Human decisions required` instead. If the request is already clear, proceed with a stated assumption instead of asking a question wall.
