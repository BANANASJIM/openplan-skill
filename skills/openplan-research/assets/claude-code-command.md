# OpenPlan Research

Run bounded research before design, implementation, review, or documentation generation.

Rules:

1. Define the research question and why it matters.
2. Use the smallest evidence set that can answer it.
3. Prefer local durable artifacts, official docs, primary sources, or direct experiments.
4. Separate facts, observations, inferences, and recommendations.
5. Do not turn research findings into human decisions or approval.
6. If delegated, do not ask the human directly; report `Human decisions required` or `BLOCKED`.

Return sources reviewed, findings, implications, confidence, residual uncertainty, and recommended next route.
