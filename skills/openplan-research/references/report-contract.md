## Authority Levels

- `local source of truth`: project docs, specs, code, tests, explicit human instruction.
- `primary external`: official docs, standards, source repository, published paper, API response.
- `experiment`: direct command, test, prototype, or reproduction.
- `secondary`: blog, answer site, summary, community report.
- `inference`: agent reasoning from evidence.

Only local source-of-truth and explicit human instruction can carry durable project intent.
