# OpenPlan Garden Checklist

## Document Identity

- What type of document is this?
- Is its authority level clear?
- Is it stable, draft, temporary, or a snapshot?
- Does the title match its purpose?
- What docs root or surface does it belong to?

## Goal/Intent Preservation

- Does it distinguish human goal/intent from agent inference?
- Are human decisions explicit?
- Are recommendations labeled as recommendations?
- Are unresolved questions still visible?

## Structure

- Does the first section orient the reader?
- Are headings meaningful and scan-friendly?
- Is there a clear path from problem to rationale to outcome?
- Is the next action visible if action is expected?

## Rationale and Evidence

- Are decisions backed by why/alternatives/consequences?
- Are claims traceable to artifacts, source statements, or evidence?
- Are missing facts labeled?
- Are assumptions separated from facts?

## Drift and Staleness

- Does the document contain dates, version claims, "current/latest", or status statements that may be stale?
- Are superseded decisions marked?
- Does it conflict with a more authoritative document?
- Does a memory entry conflict with durable docs?

## Periodic Maintenance

- Are old research, review, handoff, or temporary docs still being treated as active?
- Should any document be archived, marked stale, or promoted through record review?
- Are repeated explanations better replaced with a single source and references?
- Are TODOs, open questions, or next actions still owned and relevant?
- Would another agent know what to read first and what to ignore?
- Would another agent know which docs root is authoritative?

## Duplication and Context Weight

- Is content repeated instead of referenced?
- Could a path reference replace a copied section?
- Is the document carrying implementation detail that belongs elsewhere?
- Is it too long for its purpose?

## Zero-Context Comprehension

A future reader should answer:

- What is this?
- Why does it exist?
- What is authoritative here?
- What remains undecided?
- What should happen next?

If not, report a garden finding.
